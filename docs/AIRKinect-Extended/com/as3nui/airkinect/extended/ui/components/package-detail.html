Classes
 ClassDescription
 BaseUIComponent Foundational Class for all AIRKinect Extended UIComponents.
 CrankHandle Crank Handle Component is used to provide a unlimited range of movement through a small component.
 Handle Handle is core for all components that require snapping and capturing.
 HotSpot HotSpot component dispatches OVER and OUT events and will never be selected.
 RepeatingSelectableHandle RepeatingSelectableHandle component allows for multiple selections of the same handle.
 SelectableHandle SelectableHandle uses a Selection Timer upon capture to determine selection.
 SlideHandle SlideHandle component provides a mechanism for "Swipe" love movement with incremental updates.
 Target A Target does not attract a cursor but does use the Selection Timer Mechanism.
