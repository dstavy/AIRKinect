Classpublic class Target
InheritanceTarget HotSpot BaseUIComponent flash.display.Sprite

A Target does not attract a cursor but does use the Selection Timer Mechanism. This component dispatches OVER, OUT, and SELECTED. Upon Rollover a Target will not capture a cursor but will turn it invisible then the target will position the Selection Timer at the location of the cursor. This gives the illusion that the cursor has changed into the selection timer but still allows movement. The flow of this component would be as follows.

  • Cursor Over
  • Cursor Turned invisible
  • Selection Timer placed at location of cursor
  • Selection Timer Started
  • (optional, if moved) Cursor movement mapped to selection timer movement
  • Selection Time Complete (dispatches selected event)
  • Selection Timer removed
  • Cursor turned visible
