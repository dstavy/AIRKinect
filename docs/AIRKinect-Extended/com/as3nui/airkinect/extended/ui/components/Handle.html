Handle()Constructor
public function Handle(icon:DisplayObject, selectedIcon:DisplayObject = null, disabledIcon:DisplayObject = null, capturePadding:Number = .45, minPull:Number = .1, maxPull:Number = 1)

HAndle Component should now be instantiated but provides the core functionality for all Capturing and Snapping components.

Parameters
icon:DisplayObject — Normal Icon used for this handle
 
selectedIcon:DisplayObject (default = null) — Icon used when handle Captures the cursor
 
disabledIcon:DisplayObject (default = null) — Icon used when this handle is disabled
 
capturePadding:Number (default = .45) — Padding to add around Icon for attraction
 
minPull:Number (default = .1) — Attraction pull on the cursor when it is at the maximum distance from the center
 
maxPull:Number (default = 1) — Maximum attraction pull for this handle as the cursor moves towards snapping
