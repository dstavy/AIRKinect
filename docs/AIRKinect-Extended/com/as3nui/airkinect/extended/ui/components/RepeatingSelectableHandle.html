Classpublic class RepeatingSelectableHandle
InheritanceRepeatingSelectableHandle Handle BaseUIComponent flash.display.Sprite

RepeatingSelectableHandle component allows for multiple selections of the same handle. This is done with an optional repeat timer. The flow of this component would be as follows.
  • Cursor Attraction
  • Cursor Capture
  • SelectionTimer Start
  • SelectionTimer Complete (dispatched event)
  • (optional) Repeat Timer start
  • (optional) Repeat Timer complete
  • SelectionTimer Start
    • (repeats)
If RepeatTimer is left null the selection timer will simply start again after completed
