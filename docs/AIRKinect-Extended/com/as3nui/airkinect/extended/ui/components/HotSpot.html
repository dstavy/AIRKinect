Classpublic class HotSpot
InheritanceHotSpot BaseUIComponent flash.display.Sprite
Subclasses Target

HotSpot component dispatches OVER and OUT events and will never be selected. If provides for simple roll over selections and roll out deselection.

var s:Sprite = new Sprite(); s.graphics.beginFill(Math.random()xffffff); s.graphics.drawRect(0,0,200,200); var hotSpot:HotSpot = new HotSpot(s); hotSpot.addEventListener(UIEvent.OVER, onHotSpotOver, false, 0, true); this.addChild(hotSpot);
