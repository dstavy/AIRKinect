Classpublic class AIRKinectManager
InheritanceAIRKinectManager Object

AIRKinectManager is a singleton Class used to manage Skeletons, RGB Frames and Depth Frames. Manger should be initialized prior to use

var flags:uint = AIRKinect.NUI_INITIALIZE_FLAG_USES_SKELETON | AIRKinect.NUI_INITIALIZE_FLAG_USES_COLOR AIRKinectManager.initialize(flags);

Once Manager is initialized one can attach to any of the avaliable Signals to track events such as onSkeletonAdded, onSkeletonRemoved, onRGBFrameUpdate, etc

onDepthFrameUpdateproperty 
onDepthFrameUpdate:Signal  [read-only]

Signal that is dispatched when a new Depth Frame is ready Will be dispatched with two parameters of type BitmapData, ByteArray Bitmap data is current Depth Bitmap Data. ByteArray is only used in AIRkinectFlags.NUI_INITIALIZE_FLAG_USES_DEPTH mode. Data is a Byte Array in the format x,y,z where each is a Unsigned Short the array will contain ((_frame.width _frame.height) (2 )) bytes. (2 bytes per UShort and 3 USHORTs per pixel.)
