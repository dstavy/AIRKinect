Packagecom.as3nui.airkinect.extended.manager.gestures
Classpublic class SwipeGesture
InheritanceSwipeGesture AbstractKinectGesture Object

Swipe gesture is defined as a joint moving over a set velocity in some direction and then falling below another velocity. Swipes can be in any direction in space (left, right, forward back, up, down)
