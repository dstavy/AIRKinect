Classpublic class AbstractKinectGesture
InheritanceAbstractKinectGesture Object
Implements IKinectGesture
Subclasses ScaleGesture, SwipeGesture

Base class for Gestures to build off of. Provides all signals needed to dispatched Region support and function structure for creating gestures Most core gesture functionality is provided here and should be overridden in new gestures

public function AbstractKinectGesture(skeleton:ExtendedSkeleton, regions:Vector.<Region> = null, priority:uint = 0)

Base constructor for all gestures

Parameters
skeleton:ExtendedSkeleton — Skeleton to track
 
regions:Vector.<Region> (default = null) — regions to test for containment at start of gesture
 
priority:uint (default = 0) — priority determines if a getsure cancles other gestures. Exmaple. a gesture of priority 2 will occur and cancel any gestures of priority 1. gestures of the same priority will all get dispatched. Default is 0
