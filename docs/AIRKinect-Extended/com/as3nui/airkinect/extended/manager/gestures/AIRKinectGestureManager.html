Classpublic class AIRKinectGestureManager
InheritanceAIRKinectGestureManager Object

Gesture Managers handles updating gestures through the different states, Gestures are added and removed to this Singleton to be updated on EnterFrame. For exmaple

var leftSwipeGesture:SwipeGesture = new SwipeGesture(skeleton, SkeletonPosition.HAND_LEFT, null, true, false, false); AIRKinectGestureManager.addGesture(leftSwipeGesture);
