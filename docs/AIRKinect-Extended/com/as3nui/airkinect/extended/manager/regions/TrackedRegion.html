Packagecom.as3nui.airkinect.extended.manager.regions
Classpublic class TrackedRegion
InheritanceTrackedRegion Region Object

Tracked Region is used to attach a region relative to any joint on a skeleton Positions are then defined relative to the joints positions. For example a top of -1 and bottom of 1 attached to any joint would always contain the full height of the Kinect. a top of -.5 would always be half the height upwards from the current joint, and so on.
