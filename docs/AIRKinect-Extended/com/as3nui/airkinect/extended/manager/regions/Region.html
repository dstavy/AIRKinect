Classpublic class Region
InheritanceRegion Object
Subclasses TrackedRegion

Region is a 3D Rectangle in space. it contains a Width, Height and Depth. Added functions allow for checking if a vector 3D is contained inside of a region. Region can be created like

var frontTopLeft:Region = new Region(.3, .3, .4, .4, 1, 2, "ftl");

Region()Constructor
public function Region(top:Number, left:Number, bottom:Number, right:Number, front:Number, back:Number, id:String = null, data:* = null)

Creates a region Number below are defaults from the kinect though a region can be of any scale.

Parameters
top:Number — Position of the top of the region (0 is top, .5 is middle, 1 is the bottom)
 
left:Number — Position of the left of the region (0 is left, .5 is middle, 1 is right)
 
bottom:Number — Position of the bottom of the region (0 is top, .5 is middle, 1 is the bottom)
 
right:Number — Position of the right of the region (0 is left, .5 is middle, 1 is right)
 
front:Number — Position of the front of the region (0 is front, 2 is middle, 4 is back)
 
back:Number — Position of the back of the region (0 is front, 2 is middle, 4 is back)
 
id:String (default = null) — Optional Id for region, if blank id will be region_(count of regions)
 
data:* (default = null) — Optional Data to associate to region.
